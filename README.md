# 📊 Data Retrieval & Sorting Dashboard

A simple and interactive Student Data Dashboard built using **HTML, CSS, and JavaScript**.

This project demonstrates data retrieval, sorting, filtering, and dynamic DOM manipulation without using any backend.

---

## 🚀 Features

✅ Display student records in a table  
✅ Sort by:
- Name (Alphabetical Order)
- Joining Date (Ascending Order)

✅ Filter by Department  
✅ Display count of students per department  
✅ Fully responsive and clean UI  
✅ 100% Frontend (No Backend Required)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Array Methods (map, filter, sort)
- Event Listeners
